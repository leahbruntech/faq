About

Give your users a great experience with your website or application and save time and money in the process.


Features:

  • Nested catgeories that are easy to navigate.
  • Users can like articles and these are used to order the articles in listings and search reasults so that well liked and helpful articles are first.
  • Easy to redesign since the HTML/CSS is separated from the PHP and is built using the twitter bootstrap
  • Facebook, Twitter, and Google+ share buttons.
  • Search Engine friendly Urls.
  • Contact forms are integrated to each articles for the user to use when they need clarification.
  • All the text is in one file making this script very easy to translate.
  • You can define realtionships between articles and other articles and categories making it even easier for users tofind what they need.

Installation:

  1. Unzip the package
  2. Open the file /php/config.php with a text editor and put in the details for your server
  3. Navigate to the script install.php using your browser
  4. An admin account is created with the following credetials:
    Username: admin
    Password: pass
  5. Delete the file install.php

Getting Started:

Sign in and click on the Settings menu. All the settings have been initialized but you will need to change the time zone and admin email. Also change the password to something secure.

The script is basically a sepecialized blog or wiki. You will need to create categories by clicking in the Categories menu and then create articles to be place in these categories by clicking on the Articles menu.


Translating the script:

To translate this script from English to any other language:

  1. Open the file /str/strings.xml using a text editor
  2. You will find all the text that appears on all the page there. Simply translate them one by one

Redesigning the look of the site:

Even though the whole script is built on bootstrap, I have duplicated the css so that the admin and frontend are separate. This is ment to allow you the quickly change the frontend and not bother with the admin.

The CSS is in the folder css and the HTML is in the folder HTML. The files for the frontend start with frontend_ and for the admin start with admin_

The files that yo will want to start with are frontend_base.html and admin_base.html. Modifying the two files affects all the pages in the repective sections.
